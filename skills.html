笔杆子 ✍️ · 内容创作 · 自动化
内容全链路发布
从选题到发布，一条龙搞定。公众号文章写完自动生成封面图，自动格式化排版，一键发布到微信。支持同步到 X/Twitter Thread，适配小红书图文格式。笔杆子一个人干完原来内容团队三个人的活。
  • baoyu-url-to-markdown — 网页抓取转写作素材
  • baoyu-markdown-to-html — Markdown 转公众号 HTML
  • baoyu-cover-image — 自动生成文章封面图
  • baoyu-post-to-wechat — 一键发布到微信公众号
  • baoyu-post-to-x — 发布到 X/Twitter Thread
  • baoyu-slide-deck — 内容转演示幻灯片

产品参谋长 🔍 · 竞品分析 · 实时监控
竞品情报雷达
7×24 自动爬取竞品官网、产品更新日志、App 应用商店评论、社交媒体动态。每天自动生成情报摘要，推送到飞书或你指定的频道。参谋长睡觉时也在盯着你的竞争对手。
  • competitor-monitor — 竞品动态实时监控
  • review-analyzer — App 评论情感分析
  • market-brief — 每日市场情报摘要生成
  • trend-radar — 行业趋势关键词追踪
📦 4个技能
技术专家 ⚙️ · RAG · 知识管理
RAG 知识引擎
把你的内部文档、制度手册、会议记录、历史项目变成可检索的私有知识库。支持飞书文档、Notion、本地 PDF、Word 多源接入。员工问任何内规，AI 直接引用原文，附带页码来源。
  • doc-ingestion — 多格式文档向量化入库
  • feishu-sync — 飞书文档自动同步索引
  • rag-query — 带来源引用的精准问答
  • knowledge-update — 知识库自动增量更新
  • search-audit — 检索命中率监控与优化

技术专家 ⚙️ · Code Agent · 自动化开发
Code Agent 交付流水线
从需求描述到代码提交，技术专家全程接管。自动写代码、自动运行测试、自动修 bug、自动提 PR，遇到无法解决的问题才会叫醒你。Larry 睡着的时候，技术专家已经提了三个 PR。
  • code-generate — 基于需求自动生成代码
  • test-runner — 自动运行测试套件
  • bug-fixer — 测试失败自动分析修复
  • pr-submitter — 自动提交 Pull Request
  • code-review — 代码质量自动审查
  • dep-updater — 依赖库安全更新
  • deploy-checker — 部署后健康检查

用户洞察提炼
200 条用户反馈，人工整理要一天，参谋长 30 分钟出结论。支持问卷数据、用户访谈录音转写、App 评论批量分析。输出格式直接可用：问题分类 + 优先级排序 + 行动建议。

📦 4个技能
技术专家 ⚙️ · 本地部署 · 安全 · 系统工程
本地部署全家桶
敏感数据不出网的完整解决方案。Ollama 本地模型 + systemd 守护进程 + Cloudflare 边缘路由，一套跑通。服务器重启自动拉起，崩溃自动重启，你出差旅行 AI 团队照常运转。
  • ollama-setup — 本地 LLM 环境配置
  • systemd-deploy — Agent 服务守护进程配置
  • cloudflare-routing — Cloudflare Workers 边缘路由
  • ssl-auto-renew — SSL 证书自动续签
  • health-monitor — 服务健康状态监控
  • backup-restore — 自动备份与恢复机制
  • log-aggregation — 多 Agent 日志聚合分析
  • resource-guard — 服务器资源占用保护
📦 8个技能
总指挥（协调层）· 任务编排 · 多Agent · 工作流
多 Agent 任务编排
一个指令，多个 Agent 并行工作，结果自动汇总。支持顺序执行、并行执行、条件分支，任何环节失败自动重试或降级。Larry 管三个 Agent 的方式，不是一个个发命令，是设计工作流。
  • workflow-designer — 可视化任务编排配置
  • parallel-dispatch — 并行任务分发与汇总
  • retry-handler — 失败自动重试与降级
  • progress-tracker — 任务进度实时追踪
  • handoff-protocol — Agent 间工作交接协议
